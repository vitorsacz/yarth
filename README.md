# Yarth - Soluções Personalizadas

![Yarth Logo](./src/assets/logo-escrita-preto.svg)

**Yarth** é uma empresa líder em soluções de alto padrão para arquitetura e design de interiores, especializada em esquadrias de alumínio, vidraçaria completa, serralheria moderna e mobiliário personalizado. Sediada em Mairiporã (SP) e com atendimento em todo o Brasil, a Yarth une tecnologia, design e excelência técnica para transformar projetos residenciais, comerciais e corporativos.

## 🚀 Proposta de Valor

Oferecemos uma abordagem holística para acabamentos finos, garantindo:
- **Design Sob Medida:** Cada projeto é único e desenvolvido para atender necessidades específicas.
- **Qualidade Premium:** Uso exclusivo de materiais de altíssima resistência (alumínio, ferro, vidro temperado/laminado e madeira).
- **Expertise Técnica:** Equipe altamente qualificada para execução de projetos complexos.
- **Compromisso com Prazos:** Entregas rigorosas e acompanhamento pós-venda diferenciado.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as tecnologias mais modernas do ecossistema web:

- **[React 19](https://react.dev/):** Biblioteca para construção de interfaces dinâmicas e performáticas.
- **[Vite](https://vitejs.dev/):** Build tool de última geração para um desenvolvimento rápido e otimizado.
- **[Tailwind CSS](https://tailwindcss.com/):** Framework CSS utilitário para um design responsivo e consistente.
- **[Framer Motion](https://www.framer.com/motion/):** Biblioteca poderosa para animações fluídas e interações ricas.
- **[Lucide React](https://lucide.dev/):** Kit de ícones clean e versátil.
- **[TypeScript](https://www.typescriptlang.org/):** Tipagem estática para maior segurança e manutenibilidade do código.

## 📋 Funcionalidades do Site

- **Navegação Inteligente:** Menu fixo com efeito de transparência (glassmorphism) que reage ao scroll.
- **Seção Hero Impactante:** Destaque para as principais soluções com chamadas para ação (CTA) claras.
- **Sobre a Yarth:** História, missão, visão e valores detalhados da empresa.
- **Galeria de Mobiliário:** Exibição de peças exclusivas em vidro, ferro, alumínio e madeira.
- **Portfólio Interativo:** Filtros e visualização ampliada (Lightbox) de obras concluídas.
- **Diferenciais:** Cards informativos sobre os pilares de qualidade da marca.
- **Contato Integrado:** Links diretos para WhatsApp, e-mail e localização.

## 🔧 Estrutura do Projeto

```
/
├── src/
│   ├── assets/       # Imagens, logotipos e vetores (SVG, WebP)
│   ├── components/   # Componentes reutilizáveis (UI/Layout)
│   ├── App.tsx       # Componente principal e rotas
│   └── index.css     # Configurações globais e Tailwind
├── index.html        # Entry point HTML
├── metadata.json     # Metadados da aplicação
└── package.json      # Dependências e scripts de execução
```

## 🏁 Como Executar

1. **Instalar dependências:**
   ```bash
   npm install
   ```

2. **Iniciar servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Gerar build para produção:**
   ```bash
   npm run build
   ```
